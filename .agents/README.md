# Project Agents

These agent briefs are lightweight prompt targets for splitting work by responsibility.

- `system-designer.md`: architecture, schema, and migration planning.
- `planner.md`: proposal shaping and implementation-step planning.
- `reviewer.md`: PR-style review focused on bugs and regressions.
- `test-guardian.md`: testing strategy and deterministic coverage.
- `release-steward.md`: apply readiness, risk review, and rollout sanity checks.

Use them as specialized perspectives, not as permission to expand scope.
